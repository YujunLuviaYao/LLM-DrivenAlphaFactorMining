# Timing Files Guide

## Use These (Current)

- `final_submission_summary.csv`: final table for report submission (Baseline~Opt4)
- `opt1_timing.csv`: Opt1 single-run timing
- `opt2_timing.csv`: Opt2 single-run timing
- `opt3_timing_tuned.csv`: Opt3 tuned timing (best thread)
- `opt4_timing.csv`: Opt4 GPU timing
- `opt3_thread_sweep.csv`: Opt3 thread sweep details
- `opt3_best_thread.csv`: Opt3 best-thread record
